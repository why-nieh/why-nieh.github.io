# 21.1 信息物理系统概述
- **物联网简介**：
  - 定义：通过信息传感设备，将任何物理实体与互联网相连
  - 基本功能：信息获取、传送、处理、施效
  - 特征：整体感知、可靠传输、智能处理
- **CPS简介**：
  - 定义：计算、通信、控制（3C）有机融合，实现物理空间与信息空间实时交互
  - 闭环赋能体系：状态感知、实时分析、科学决策、精准执行
- **CPS与物联网关系**：CPS是物联网的演进，更强调反馈控制

# 21.2 信息物理系统架构
- **CPS分层架构**：
  - 单元级（最小单元，智能轴承、智能机器人）
  - 系统级（多个单元通过网络互联，生产线）
  - 系统之系统级（SoS，多个系统通过平台互联，企业级）
- **CPS典型系统构成**：
  - 传感器、执行器、决策控制单元
  - 逻辑分层：决策层、网络层、物理层
- **CPS特点**：
  - 全局虚拟、局部物理
  - 深度嵌入
  - 事件驱动
  - 以数据为中心
  - 时间关键
  - 安全关键
  - 异构
  - 高可信赖
  - 高度自主
  - 领域相关性

# 21.3 信息物理系统技术框架
- **核心技术**：
  - 感知和自动控制（智能感知、虚实融合控制）
  - 工业软件（嵌入式软件、MBD、CAX/MES/ERP）
  - 工业网络（现场总线、工业以太网、无线、SDN）
  - 工业云和智能服务平台（边缘计算、雾计算、大数据分析）
- **支撑技术**：
  - 计算理论：普适计算、嵌入式计算、分布式计算、云计算、移动计算、自律计算、可信计算
  - 网络传输：IPv6、NGN/5G
  - 传感器网络：WSN、WSAN
- **协议标准**：
  - 协议栈模型（OSI、5层）
  - 相关协议分类：近距离（NFC、RFID、Bluetooth）、远距离蜂窝（2G/3G/4G/5G）、远距离非蜂窝（ZigBee、Wi-Fi、LoRa）、有线（Ethernet、RS-232、RS-485、USB）
  - 具体协议详解：
    - RFID：电子标签+阅读器
    - NFC：被动/主动模式，点对点、读卡器、模拟卡片
    - Bluetooth/BLE：低功耗
    - ZigBee：低功耗、自组织、Mesh
    - LoRa/LoRaWAN：长距离、低功耗，Class A/B/C
    - LPWAN：低功耗广域网，NB-IoT、LoRa、SigFox
    - 蜂窝移动通信：GSM、GPRS、3G/4G/5G（eMBB、mMTC、URLLC）
    - Ethernet：IEEE802.3，Cat5/6，光纤
    - RS-232/485：串行通信
    - USB：通用串行总线
    - AMQP：高级消息队列协议
    - CoAP：受限应用协议
    - DDS：数据分发服务
    - LwM2M：轻量机器间通信
    - MQTT：消息队列遥测传输，QoS0/1/2，持久性
    - Wi-Fi：IEEE802.11a/b/g/n/ac/ax，基础结构/自组织模式
    - XMPP：可扩展消息传送和存在状态协议
    - Z-Wave：低功耗Mesh，主控+从设备
    - MBus：仪表总线

# 21.4 信息物理系统开发技术
- **感知和控制相关技术**：
  - 智能感知：传感器、RFID
  - 虚实融合控制：嵌入控制、虚体控制、集控控制、目标控制（四层）
- **工业软件相关技术**：
  - 嵌入式软件
  - MBD（基于模型定义）：三维数字化产品描述
  - CAX/MES/ERP：CAD/CAE/CAM，制造执行系统，企业资源计划
- **网络相关技术**：
  - 现场总线
  - 工业以太网（TSN）
  - 无线技术（WirelessHART、ISA100.11a、Wi-Fi、ZigBee、5G）
  - SDN（软件定义网络）
- **服务平台相关技术**：
  - 边缘计算（靠近数据源处理）
  - 雾计算（分布式，本地设备处理）
  - 大数据分析（产品创新、故障预测、供应链优化）

# 21.5 控制系统与网络通信
- **自动控制系统行为模式**：
  - 开环控制
  - 闭环控制（反馈）
  - 顺序控制
  - 批量控制
- **控制系统通信要求**：
  - 周期性、确定性
  - 通信模式：确定性周期（DPC）、确定性非周期（DAC）、非确定性通信（NDC）

# 21.6 CPS应用分析与设计
- **工业设计系统**：
  - 需求：设计过程缺乏真实数据
  - 解决方案：通过CPS采集真实数据，仿真优化设计（产品及工艺设计、生产线设计）
- **生产制造系统**：
  - 需求：设备分散、协同性差、管理困难
  - 解决方案：设备管理（互联互通）、生产管理（数据驱动决策）、柔性制造（快速响应变化）
- **产品服务系统**：
  - 需求：装备复杂、维护困难
  - 解决方案：健康管理（状态监测、评估）、智能维护（预测性维护）、远程诊断（专家远程）、协同优化（数据共享）、共享服务（群体智能）
- **产业链协同系统**：
  - 需求：设计者、生产者、使用者协同
  - 解决方案：CPS实现全产业链信息融合，无人装备、产业链互动
