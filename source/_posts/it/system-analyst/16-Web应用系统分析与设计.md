# 16.1 Web应用系统简介
- **Web应用程序**：运行在浏览器中的软件，优势（易于访问、开发高效、使用简单、扩展性强）
- **Web应用特征**：
  - 云托管和可扩展性
  - 跨平台（Linux、macOS、Windows）
  - 自动化测试
  - 支持传统应用（服务器端渲染）和SPA（单页应用）
  - 简单开发和部署

# 16.2 Web应用架构设计
- **设计原则**：
  - 分离关注点
  - 封装
  - 依赖关系反转
  - 显式依赖关系
  - 单一责任
  - 避免自我重复
  - 持久性无知（POCO）
  - 有界上下文（领域驱动设计）
- **架构分类**：
  - 整体式架构（单体）
  - 微服务架构（独立服务）
  - 无服务器架构（函数计算）
- **架构模式**：
  - **B/S架构**：三层（表现层、逻辑层、数据层）
  - **P2P架构**：对等网络
  - **MVC/MVP**：
    - MVC：Model、View、Controller
    - MVP：Model、View、Presenter（解耦View和Model）
  - **微服务架构**：
    - 服务注册与发现
    - 服务监控
    - 服务容错（熔断、限流、降级）

# 16.3 Web应用开发框架
- **Java EE框架**：
  - SSH：Spring（IoC、AOP）、Struts（MVC）、Hibernate（ORM）
  - JSF（Java Server Faces）：组件化
- **.NET框架**：ASP.NET、CLR
- **Web层开发框架**：
  - WebPage 3.0（可视化）
  - AJAX框架：Prototype、jQuery、Mootools、DOJO、Ext JS、AFLAX

# 16.4 Web应用系统开发
- **通信协议**：
  - HTTP（无状态，会话跟踪：URL重写、Cookie）
  - RTP/RTSP（流媒体）
  - SMTP/POP3（邮件）
- **数据存储**：
  - 缓存系统（应用服务器缓存、全局缓存、分布式缓存、CDN）
  - 云存储（Amazon S3、Azure Storage、Google Cloud Storage）
  - 负载均衡器（四层LVS、七层Nginx、硬件F5）
  - 消息队列（Kafka、RabbitMQ）
- **客户端技术**：
  - HTML/HTML5（新元素、语义）
  - XML（可扩展标记语言）
  - DHTML（动态HTML）
  - CSS（层叠样式表）
  - Flash/Flex（矢量动画）
  - DOM（文档对象模型）
  - JavaScript/AJAX（异步交互）
- **服务器端技术**：
  - CGI（公共网关接口）
  - PHP（超文本预处理器）
  - ASP/ASP.NET（微软）
  - Servlet（Java）
  - JSP（Java Server Pages）
  - Perl/Ruby/Python（脚本语言）
- **系统部署**：
  - 部署粒度：细粒度（单个组件）、打包发布
  - 部署原则：管理期望、测试、建立支持、先改错、灰度发布
  - 部署环境：操作系统（Windows、Linux）、应用服务器（Apache、IIS、WebLogic、Websphere）、负载均衡
  - 版本控制和CMS（内容管理系统）

# 16.5 Web应用系统测试
- **测试概述**：与传统测试相比的挑战（用户量大、链接复杂、多层架构、浏览器差异、导航功能、技术多样）
- **测试过程**：需求分析、策略、环境、用例、执行、报告
- **功能测试**：
  - 链接测试（所有链接是否正常）
  - 表单测试（输入验证、默认值、异常）
  - 数据校验（合法性、完整性）
  - Cookies测试（存储、有效期、加密）
  - 数据库测试（数据完整性、输出错误）
- **性能测试**：
  - 速度测试（网络连接、业务处理）
  - 负载测试（并发用户数）
  - 压力测试（超出负载）
  - 强度测试（极限状态）
  - 并发测试
  - 大数据量测试
  - 配置测试
  - 可靠性测试
  - 性能测试方法：虚拟用户、WUS
- **安全性测试**：
  - 数据加密测试
  - 用户身份验证测试
  - 日志文件测试
  - Session测试
  - 备份与恢复测试
  - 访问控制策略测试
  - 安全漏洞测试（XSS、SQL注入）
  - TCP端口测试
  - 服务器端脚本漏洞检查
  - 防火墙测试
- **Web服务测试**：
  - 特性：参与者分布、远程、黑盒、在线、动态绑定
  - 内容：基础设施验证、独立测试、集成测试
