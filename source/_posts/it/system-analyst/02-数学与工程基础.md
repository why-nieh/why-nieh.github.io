# 2.1 数学统计基础
- **概率论基础**：
  - **事件运算**：和事件$(\cup)$、积事件$(\cap)$、差事件（−）、逆事件（¯）、互斥事件。
  - **概率性质**：
    - $P(\emptyset)=0, P(\Omega)=1$
    - $0\le P(A)\le 1$
    - $P(\overline{A})=1-P(A)$
    - $P(A-B)=P(A)-P(AB)$
    - 若 $B\subseteq A$，则 $P(A-B)=P(A)-P(B)$
  - **条件概率**：$P(B|A)=\frac{P(AB)}{P(A)}$
  - **独立性**：$P(AB)=P(A)P(B)$
  - **加法公式**：
    - $P(A\cup B)=P(A)+P(B)-P(AB)$
    - $P(A\cup B\cup C)=P(A)+P(B)+P(C)-P(AB)-P(AC)-P(BC)+P(ABC)$
  - **乘法公式**：
    - $P(AB)=P(A)P(B|A)$
    - $P(A_1A_2...A_n)=P(A_1)P(A_2|A_1)...P(A_n|A_1...A_{n-1})$
  - **全概率公式**：$P(A)=\sum_{i=1}^n P(B_i)P(A|B_i)$，其中 $B_i$ 是完全事件组。
  - **贝叶斯公式**：$P(B_k|A)=\frac{P(B_k)P(A|B_k)}{\sum_i P(B_i)P(A|B_i)}$
  - **伯努利二项概率**：$P_n(k)=C_n^k p^k (1-p)^{n-k}$

- **随机变量及其分布**：
  - **离散型**：概率分布 $p_k=P(\xi=x_k)$，性质 $p_k\ge0,\sum p_k=1$
  - **连续型**：密度函数 $p(x)$，分布函数 $F(x)=\int_{-\infty}^x p(t)dt$，性质 $p(x)\ge0,\int_{-\infty}^{\infty}p(x)dx=1$
  - **二维离散型**：联合概率 $p_{ij}=P(\xi=x_i,\eta=y_j)$，边际分布 $p_{i\cdot}=\sum_j p_{ij}$，$p_{\cdot j}=\sum_i p_{ij}$
  - **二维连续型**：联合密度 $p(x,y)$，分布函数 $F(x,y)=\int_{-\infty}^y\int_{-\infty}^x p(u,v)dudv$

- **数字特征**：
  - **数学期望**：
    - 离散：$E\xi=\sum x_k p_k$
    - 连续：$E\xi=\int_{-\infty}^{\infty} x p(x)dx$
    - 性质：线性 $E(k\xi)=kE\xi$，$E(\xi+\eta)=E\xi+E\eta$，独立时 $E(\xi\eta)=E\xi\cdot E\eta$
  - **方差**：
    - 离散：$D\xi=\sum (x_k-E\xi)^2 p_k$
    - 连续：$D\xi=\int_{-\infty}^{\infty} (x-E\xi)^2 p(x)dx$
    - 性质：$D(k\xi)=k^2 D\xi$，独立时 $D(\xi\pm\eta)=D\xi+D\eta$，$D\xi=E(\xi^2)-(E\xi)^2$

- **常用分布**：
  - **0-1分布**：$P(X=1)=p, P(X=0)=1-p$，$E=p$，$D=p(1-p)$
  - **二项分布 $B(n,p)$**：$P(X=k)=C_n^k p^k (1-p)^{n-k}$，$E=np$，$D=np(1-p)$
  - **几何分布 $G(p)$**：$P(X=k)=p(1-p)^{k-1},k=1,2,...$，$E=\frac{1}{p}$，$D=\frac{1-p}{p^2}$
  - **泊松分布 $P(\lambda)$**：$P(X=k)=\frac{\lambda^k}{k!}e^{-\lambda}$，$E=\lambda$，$D=\lambda$
  - **均匀分布 $U(a,b)$**：密度 $p(x)=\frac{1}{b-a}, a\le x\le b$，$E=\frac{a+b}{2}$，$D=\frac{(b-a)^2}{12}$
  - **标准正态分布 $N(0,1)$**：密度 $\phi(x)=\frac{1}{\sqrt{2\pi}}e^{-x^2/2}$，$E=0$，$D=1$

- **统计分析方法**：
  - **常用统计量**：
    - 样本均值 $\overline{X}=\frac{1}{n}\sum X_i$
    - 样本方差 $S^2=\frac{1}{n-1}\sum (X_i-\overline{X})^2$（修正），$B^2=\frac{1}{n}\sum (X_i-\overline{X})^2$
    - 样本标准差 $S=\sqrt{S^2}$
    - 样本k阶原点矩 $A_k=\frac{1}{n}\sum X_i^k$
    - 样本k阶中心矩 $B_k=\frac{1}{n}\sum (X_i-\overline{X})^k$
    - 次序统计量
  - **参数估计**：
    - 点估计：矩估计法（用样本矩估计总体矩）、极大似然估计（使似然函数最大）
    - 区间估计：置信区间、置信水平
  - **假设检验**：
    - 参数假设检验：U检验、t检验、χ²检验、F检验
    - 显著性检验
  - **回归分析**：
    - 一元线性回归 $y = a + bx$，最小二乘法求参数
    - 多元线性回归
    - 模型检验、变量筛选（逐步回归等）
  - **方差分析**：
    - 单因素方差分析：比较多个均值
    - 双因素方差分析：无交互作用、有交互作用
    - 前提：可比性、正态性、方差齐性
  - **正交试验法**：
    - 正交表 $L_n(m^k)$，均衡分散、整齐可比
    - 极差分析、方差分析

# 2.2 图论应用
- **最小生成树**：
  - **普里姆算法**：
    1. 初始 $U=\{v_0\}, T=\emptyset$
    2. 重复：选择最小权边 $(u,v)$，$u\in U, v\notin U$，将 $v$ 加入 $U$，边加入 $T$
    3. 直到 $U=V$
    - 时间复杂度 $O(n^2)$，适合稠密图。
  - **克鲁斯卡尔算法**：
    1. 初始森林 $T=(V,\emptyset)$
    2. 按权值递增顺序选择边，若两端点在不同连通分量，则加入，否则舍弃
    3. 直到有 $n-1$ 条边
    - 时间复杂度 $O(e\log e)$，适合稀疏图。
  - **博鲁夫卡算法**：
    1. 每个节点作为连通分量
    2. 对每个分量找最小权边连接其他分量，加入
    3. 重复直到连通
    - 时间复杂度 $O(E\log V)$，适合并行。
  - **反向删除算法**：
    1. 从大到小排序边
    2. 依次尝试删除边，若删除后图不连通，则保留
    - 时间复杂度 $O(E\log V)$

- **最短路径**：
  - **迪杰斯特拉算法**：
    - 单源，非负权
    - 步骤：
      1. 初始化 $dist[s]=0$，其他 $+\infty$，集合 $S=\emptyset$
      2. 从 $V-S$ 中选最小 $dist$ 的点 $u$，加入 $S$
      3. 对 $u$ 的邻接点 $v$，更新 $dist[v]=\min(dist[v], dist[u]+w(u,v))$
      4. 重复2-3直到所有点加入
    - 时间复杂度 $O(n^2)$，可用堆优化到 $O((n+e)\log n)$
  - **弗洛伊德算法**：
    - 动态规划，所有点对
    - 初始化距离矩阵 $d[i][j]=w(i,j)$（无边为无穷）
    - 对于 $k=1..n$，更新 $d[i][j]=\min(d[i][j], d[i][k]+d[k][j])$
    - 时间复杂度 $O(n^3)$
  - **贝尔曼-福特算法**：
    - 单源，可负权，检测负环
    - 步骤：进行 $n-1$ 轮松弛（每条边尝试更新），第 $n$ 轮若还能更新则有负环
    - 时间复杂度 $O(ne)$
  - **A*算法**：
    - 启发式搜索，估计函数 $f(n)=g(n)+h(n)$
    - 使用优先队列，优先扩展 $f$ 最小的节点
    - 时间复杂度依赖于启发函数
  - **约翰逊算法**：
    - 对所有边重新赋权，消除负权
    - 先添加虚拟节点0，用Bellman-Ford求势能 $h(v)$
    - 边权更新为 $w'(u,v)=w(u,v)+h(u)-h(v)$
    - 对每个节点运行Dijkstra
    - 时间复杂度 $O(VE\log V)$
  - **SPFA算法**：
    - 队列优化的Bellman-Ford
    - 步骤：初始源点入队，松弛邻接点，若更新且不在队中则入队
    - 平均快，最坏 $O(VE)$

- **网络最大流**：
  - **增广路算法**（Ford-Fulkerson）：
    - 寻找增广路径，增加流量，更新剩余网络
    - Edmonds-Karp：BFS找最短增广路，复杂度 $O(VE^2)$
  - **Dinic算法**：
    - 分层图 + 阻塞流，复杂度 $O(V^2E)$
  - **最小割最大流定理**：最大流值等于最小割容量。

# 2.3 预测与决策
- **预测方法**：
  - **时间序列分析**：
    - 一次指数平滑：$S_t = \alpha y_t + (1-\alpha)S_{t-1}$，预测 $\hat{y}_{t+1}=S_t$
    - 选择 $\alpha$ 使预测误差最小。
  - **因果预测**：回归分析、计量模型。
  - **定性预测**：德尔菲法、专家判断。
- **决策过程**：
  - **情报活动**：识别问题、收集信息、设定目标。
  - **设计活动**：拟定备选方案、预测后果。
  - **抉择活动**：评估方案、选择最优或满意方案。
  - **实施活动**：执行、跟踪、修正。
- **决策方法**：
  - **线性规划**：目标函数 $z=c^Tx$，约束 $Ax\le b, x\ge0$。
    - 图解法（二维）、单纯形法（多维）。
    - 解的情况：唯一最优、无穷多最优、无界解、无可行解。
  - **成本效益分析**：比较不同方案的效益与成本。
  - **效用分析**：将效用函数量化。
  - **模拟仿真**：建立模型，多次运行统计结果。
  - **决策树分析**：画出决策树，计算期望值，选择最大期望方案。

# 2.4 数学建模
- **数学模型分类**：
  - 按应用领域：人口、交通、环境等。
  - 按数学方法：初等模型、微分方程、统计回归、数学规划等。
  - 按表现特征：确定性/随机性、静态/动态、线性/非线性、离散/连续。
  - 按建模目的：描述、预报、优化、决策、控制。
  - 按对结构了解程度：白箱、灰箱、黑箱。
- **建模过程**：
  1. 模型准备：了解背景、明确目的、收集信息。
  2. 模型假设：简化问题，抓住本质。
  3. 模型建立：选择数学工具，建立数学结构。
  4. 模型求解：数学计算、软件求解。
  5. 模型分析：误差、灵敏性、稳健性分析。
  6. 模型检验：与实际比较，验证有效性。
  7. 模型应用：用于解释、预测、决策。
- **建模方法**：
  - 直接分析法
  - 类比法
  - 数据分析法
  - 构想法

# 2.5 工程伦理
- **技术与工程的区别**：技术是发明、方法、技巧；工程是建造、产品、活动。
- **主要工程伦理问题**：
  - 技术伦理：技术本身是否有善恶？工程中技术选择涉及伦理判断。
  - 利益伦理：协调各方利益，兼顾公平。
  - 责任伦理：工程师对公众、社会、自然的责任。
  - 环境伦理：减少环境影响，可持续发展。
- **处理原则**：
  - 人道主义：尊重人、不伤害。
  - 社会公正：公平分配、兼顾弱势群体。
  - 人与自然和谐发展：尊重自然规律。
- **工程师职业伦理**：
  - 忠诚责任（对雇主）
  - 社会责任（对公众）
  - 自然责任（对环境）
  - 职业伦理章程：将公众安全、健康、福祉放在首位。
