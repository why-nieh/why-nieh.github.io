# 10.1 系统规划概述
- **系统规划需求**：解决系统建设与企业目标不匹配、周期长、人员素质低、技术方案不合理、标准混乱、资金不足等问题。
- **系统规划主要步骤**：
  1. 初步调查
  2. 确定系统目标
  3. 分析子系统组成和功能
  4. 拟定实施方案
  5. 可行性研究
  6. 制定系统建设方案

# 10.2 项目的提出与选择
- **立项目标和动机**：
  - 基础研究（探索性）
  - 应用研发（面向产品）
  - 提供技术服务（咨询、集成）
  - 产品使用者（采购、定制）
- **项目立项的价值判断**：不同角色（研究、开发、服务、使用）侧重点不同
- **项目的选择和确定**：
  - 选择有核心价值的项目（与核心业务相关）
  - 评估项目约束、风险、成本、效益
  - 优先级排序（净现值、投资回收期）
  - 评估实施方式（自研、外包、购买）
  - 平衡选择合适方案（技术风险、用户锁定、扩展性、目标偏离）

# 10.3 系统分析概述
- **系统分析任务**：理解用户需求，表达为系统需求规格说明书
- **系统分析难点**：
  - 分析师与用户理解不同
  - 沟通困难
  - 环境不断变化

# 10.4 问题分析
- **研究问题领域**：上下文图（环境图），确定边界、外部实体、过程
- **分析问题和机会**：因果分析，问题/机会/目标/约束矩阵
- **制定系统改进目标**：精确、可度量、受约束（进度、成本、技术、政策）
- **汇报调查结果和建议**：书面报告提纲

# 10.5 业务流程分析
- **业务流程分析步骤**：调查、描述、确认、分析、发现问题、提出优化方案
- **业务流程分析方法**：
  - 价值链分析法
  - 客户关系分析法
  - 供应链分析法
  - 基于ERP的分析法
  - 业务流程重组
- **业务流程分析工具**：
  - **TFD（业务流程图）**：
    - 符号：业务处理单位、处理描述、输出信息、存储文件、收集资料、矢量连线
    - 绘制：内部实体、外部实体
  - **BAM（业务活动图）**：
    - 符号：行为圆圈、决策菱形、编号
    - 应用：识别业务活动，分解层次
- **业务流程建模（BPM）**：
  - **方法**：
    - 标杆瞄准（Benchmarking）
    - DEMO（动态本质建模法，业务事务三阶段）
    - Petri网（图形+数学，位置、变迁、令牌）
    - 业务流程建模语言：
      - BPEL（可执行XML）
      - BPML（元语言）
      - XPDL（流程定义交换）
      - BPMN（图形化，可转BPEL）
      - UML活动图
    - 基于服务的BPM（服务封装流程）

# 10.6 数据与数据流程分析
- **数据流内涵**：数据源、传输、转换、存储、输出
- **逻辑数据流与规则**：数据处理的逻辑规则
- **数据流的守恒**：输入 = 输出（或处理后 = 输入 + 增量，或处理前 = 处理后 + 丢失量）
- **数据流图（DFD）**：
  - 符号：数据流、进程、数据存储、外部实体
  - 层次：顶层图（上下文图）、0层图、子图
  - 绘制步骤：画输入输出、画内部、命名、检查和修改（数据守恒、父图子图平衡）
- **物理数据流图**：技术设计决策，作为构造蓝图
- **数据汇总分析**：
  - 步骤：分类编码、自顶向下整理、分类原始/最终数据、确定字长精度
  - CU矩阵：过程/数据矩阵
- **数据属性分析**：
  - 静态分析：类型、长度、取值范围、业务量、重要程度
  - 动态分析：固定值、固定个体变动、随机变动
  - 数据存储分布：本地、网络服务器
- **数据流程分析**：
  - 结构化分析（SA）：自顶向下分解，DFD
  - 面向对象方法：对象封装数据，交互传递

# 10.7 系统可行性分析
- **可行性评价准则**：
  - 经济可行性（成本效益分析）
  - 技术可行性（技术、资源、目标）
  - 法律可行性（政策、法律、道德）
  - 用户使用可行性（管理、运行）
  - 进度可行性
- **可行性研究步骤**：
  1. 复查系统目标和规模
  2. 分析现有系统
  3. 导出新系统高层逻辑模型（上下文图、E-R模型、用例模型、领域模型、IPO图）
  4. 用户复核
  5. 提出并评价解决方案
  6. 确定最终推荐的解决方案
  7. 草拟开发计划
  8. 编制和提交可行性研究报告
- **可行性研究报告**：正文格式（GB/T8567—2006），论证会流程

# 10.8 成本效益分析
- **成本和收益**：
  - 成本分类：按投资时间（基础建设、一次性、非一次性）、按成本性态（固定、变动、混合）
  - 收益分类：有形（一次性、非一次性）、无形
  - 盈亏临界分析：盈亏平衡点销售量/销售额
- **净现值分析**：
  - 货币时间价值：复利 \(F=P(1+i)^n\)，现值 \(P=F/(1+i)^n\)
  - 净现值 \(NPV = \sum (CI-CO)_t / (1+i)^t\)
  - 净现值率 \(NPVR = NPV/P\)
- **投资回收期**：
  - 静态：累计净现金流量开始为正的年份 -1 + 上年累计/当年净流量
  - 动态：满足累计折现净现金流为零的时间
  - 投资回收率 = 1/动态投资回收期
  - 投资收益率 = 投资收益/投资成本 = 1 + NPVR

# 10.9 系统方案建议
- **候选方案评价**：
  - 候选系统方案矩阵（特征对比）
  - 可行性分析矩阵（准则、权重、评分）
- **系统建议方案报告**：内容（前置部分、系统概述、研究方法、候选方案、建议方案、结论、附录）
